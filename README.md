# Shopify Foresight — Feature Proposal

A self-initiated product case study exploring a native intelligence layer for Shopify Admin. Built as part of my application to Shopify's product internship.

## What's inside

| File | Description |
|------|-------------|
| `index.html` | Main proposal — gap analysis, feature concept, mockup, competitive landscape |
| `prd.html` | Product Requirements Document — user stories, requirements, metrics, roadmap |

## The idea

Shopify's analytics show merchants what already happened. Foresight is a concept for a native layer that surfaces what to pay attention to *today* — proactive alerts, plain-English Q&A on store data, and cross-merchant benchmarks that only a platform at Shopify's scale could provide.
